# Persona-5-Strikers-Scramble-Save-Editor
Persona 5 Strikers &amp; Scramble Save Editor with Python

Works with Save Slot 1 to 9

Features
* Change Name
* Money
* Persona Points
* Bonds
* Weapons
* Armors
* Accessories
* Consumables
* Recipes
* Cash Mats.
* Skill Cards
* Persona

Run the script
```python persona5strikers_1_0_0.py```
